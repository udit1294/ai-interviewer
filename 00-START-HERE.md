## 🚀 Getting Started

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Setup Environment
```bash
cp .env.local.example .env.local
# Edit .env.local and add your Anthropic API key
```

### Step 3: Run Development Server
```bash
npm run dev
```

### Step 4: Open in Browser
```
http://localhost:3000
```

**That's it!** The app is ready to use.

## 🔧 Customization Guides

### Change Interview Questions Count
Edit: `components/ChatInterview.tsx`
```typescript
const maxQuestions = 5  // Change to desired number
```

### Add Job Roles
Edit: `components/RoleSelection.tsx`
```typescript
const commonRoles = [
  'Your New Role',
  // ... existing roles
]
```

### Modify AI Prompts
Edit: `lib/promptTemplates.ts`
All prompt functions can be customized here.

### Update Report Layout
Edit: `components/InterviewReport.tsx`
Modify JSX and Tailwind classes.

### Change Styling
Edit: `styles/globals.css` and component files
Use Tailwind CSS classes.
